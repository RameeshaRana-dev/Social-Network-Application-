# 🌐 Social Networking App
A desktop-based Social Networking Application developed in C++ using SFML.
The project combines object-oriented backend logic with an SFML-based graphical user interface.
It simulates core social media functionalities including posts, likes, comments, memories, timelines, pages, and friend interactions.
---
## ✨ Features 
- Home Feed
- Timeline View
- User Profile
- Like and Comment System
- Duplicate like prevention
- Empty Comments prevention
- Empty Post prevention
- Error popup handling
- Invalid date validation
- Share Memory Feature
- Image handling logic implemented
- Friend List
- Liked Pages
- View Page Posts
- View Liked List
- Switch between different users
- Set custom system date
- Exit 
- Dark/Light Theme Switching
